# MH-SCRIPTWRITER

## Overview
This skill executes the primary workflow for creating video scripts for Marcus Halawi's personal brand series. It involves extracting the core theme via an interview, and then meticulously drafting the script based on specific narrative frameworks and the brand's DNA.

## Prerequisites
Before beginning any work, you **MUST** internalize the Brand Voice:
- **Read `resources/MARCUS-VOICE.md`**
This document is the DNA of the script. It dictates exactly how Marcus speaks, sounds, and thinks on camera. All generated content must match this voice perfectly.

## Step 1: Interview and Outlining (Phase A)
1. Ask the user if they want to proceed with a structured interview to extract the video theme, OR if they prefer to upload a file with base info, a past script, or an outline.
2. Read `resources/interview-to-outline.md` to adopt the correct mindset and understand the parameters for the outlining phase.
3. If the user chooses the interview, engage them in a structured, sequential interview (asking one question at a time) to extract the core content. If they provide a file, review and extract the core content from the provided material.
4. Once the information is gathered (via interview or file upload), synthesize it into a structured video outline as described in the interview-to-outline document.

## Step 2: Script Drafting (Phase B)
1. Read `resources/king-framework.md` and `resources/dicks-framework.md` to understand the structural and narrative frameworks required for the script.
2. Read `resources/script-writer.md` to internalize the step-by-step script drafting process.
3. Using the approved outline from Step 1, draft the full video script based on these frameworks.
4. **CRITICAL:** Throughout the drafting process, you must continually reference `resources/MARCUS-VOICE.md` to ensure the delivery, pacing, and tone are authentically Marcus.
