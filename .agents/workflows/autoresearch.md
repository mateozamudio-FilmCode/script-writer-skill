# AutoResearch Skill Workflow

When the user runs `/autoresearch`, follow these steps to use the AutoResearch skill:

1. Read the core skill instructions located at `.gemini/skills/autoresearch/SKILL.md`.
2. Based on the user's request, determine which of the three modes they need (Discovery, Management, or Analysis).
3. If more specific instructions are needed for the mode, read the corresponding guide (`discovery_guide.md`, `management_guide.md`, or `analysis_guide.md`) in the `.gemini/skills/autoresearch/` directory.
4. Execute the chosen mode precisely as described in the skill documentation, guiding the user through the process.
